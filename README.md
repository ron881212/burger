## Eat the burger is a way the order and eat burgers with leaving the house
- Uses POST and GET request 
- Saves data on sql jawsdb
- Rendered with Handlebars   
### Eat the burger uses 2 node modules 
- express
- express-handlebars
- body-parser
- mysql
